# Internationalization (i18n)

Qleaner uses **Paraglide-JS** for Svelte 5 translation bindings.

To add a new language:
1. Copy `messages/en.json` to `messages/{code}.json` in your localized language.
2. Ensure you run `bun run build` so that the Paraglide compiler generates the necessary TS definitions.
3. Submit a PR. All components natively consume `m.*()` functions linked directly to the paraglide artifacts.
